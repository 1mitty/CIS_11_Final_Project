# LC - 3 Test Score Calculator

This project is an LC-3 assembly language program. This program prompts the user to enter five test score, this then stores them in an array, and calculates the minimum, maximum, and average score. The program will also display the letter grade equivalent for each result.

- Accepts 5 user-entered test scores
- Stores scores in an rray
- Calculates:
- Minimum score
- Maximum score
- Average score
- Displays letter grades:
- 90-100 = A
- 80-89 = B
- 70-79 = C
- 60-69 = D
- Below 60 = F
- Uses LC-3 subroutines

  Test Values used:
  52, 87, 96, 79, 61
